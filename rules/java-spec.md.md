```makedown
# Java 编码规范（强制）
## 命名规范
- 类名：大驼峰 PascalCase，名词
- 方法名：小驼峰 camelCase，动词开头
- 变量名：小驼峰，禁止拼音、单字符
- 常量：全大写 + 下划线
- 包名：全小写

## 格式规范
- 缩进 4 空格，禁用 Tab
- 大括号 { 不换行
- 运算符两侧加空格
- 一行不超过 120 字符
- 方法间空一行

## 代码规范
- 禁止魔法值，必须用常量
- 集合初始化指定大小
- 必须判空，防止 NPE
- 禁止 e.printStackTrace()
- 禁止循环内 DB / 远程调用
- 统一返回 Result 结构

## 注释规范
- 类必须加 Javadoc
- public 方法必须注释
- 复杂逻辑必须加行内注释
```
