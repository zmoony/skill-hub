```txt

MyBatis-Plus 3.5.4.1、PostgreSQL 15.6、nacos 2.2.3、OpenFeign 3.1.5
SpringSecurity + JWT、Knife4j 4.5.0、SLF4J + Logback
Lombok 1.18.30、Hutool 5.8.23、Druid 连接池

要求严格遵守：
1. 标准分层：controller / service / impl / mapper / entity / config / exception / util / constant
```

### 2. Codex 顶级 Code Review 万能指令（最强审查模板）
```text
请对以下 Java 后端代码做生产级 Code Review，严格检查以下内容：

1. 空指针风险（NPE）
2. 未处理异常、try-catch 不合理、吞异常
3. 参数校验缺失、越权风险
4. 事务是否正确、数据库操作是否安全
5. MyBatis-Plus 滥用、全表更新/删除风险
6. 代码规范、命名、注释、结构清晰度
7. 业务逻辑边界 case 是否覆盖
8. 性能隐患、慢 SQL 风险
9. 代码冗余、可优化点
10. 安全风险（SQL 注入、权限绕过）

输出格式：
1. 问题清单（行号+问题+风险等级）
2. 优化建议（每条对应修改方案）
3. 修复后的完整代码
4. 总体评分（100 分制）
```
