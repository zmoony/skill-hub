```md
# 技能：生成 Swagger/Knife4j 接口文档
## 描述
自动给 Controller 生成标准接口文档，包含：
@ApiOperation、@ApiParam、VO/DTO 字段注解
自动生成请求示例、响应示例、错误码说明

## 规范
- 所有接口必须加 @ApiOperation("接口说明")
- 入参加 @ApiModelProperty("字段说明")
- 必传字段标注 required = true
- 响应类统一加注释
- 接口分组：模块名
```
